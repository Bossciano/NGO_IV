## Setup — 2 steps

### Step 1: Download real images
Open Terminal in this folder and run:
```bash
chmod +x download-images.sh
./download-images.sh
```
Windows users — use Git Bash, WSL, or download each URL manually.

### Step 2: Open or host
- **Local preview:** open `index.html` in Chrome/Firefox
- **Hosting:** Upload the entire folder to Netlify, Vercel, cPanel, or any web host

## Hosting on Netlify (free, easiest)
1. Go to netlify.com → "Add new site" → "Deploy manually"
2. Drag the entire `cysd-website` folder into the deploy box
3. Done — your site is live in seconds
